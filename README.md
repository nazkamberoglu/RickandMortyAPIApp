Rick and Morty API Application

This project is a web application that consumes the Rick and Morty API to fetch and display character and episode information from the animated series.

Description

The application retrieves character and episode data from the Rick and Morty API and presents it to users through a clean and interactive user interface. Users can browse characters and manage their favorite characters by adding or removing them from a favorites list.

Features

Display character information from the Rick and Morty API

Fetch and show episode data

Add and remove favorite characters

User-friendly and responsive interface

Real-time API data consumption

Technologies Used

Rick and Morty API

ASP.NET / JavaScript

HTML, CSS, Bootstrap

Purpose

This project was developed to practice API integration, dynamic data rendering, and user interaction in a web application.
